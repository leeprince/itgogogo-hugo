物联网

感觉现在已经是万物互联的时代了，未来物联网+5G、物联网+AI？

大家好，我是皇子。

17年最火的ofo共享自行车，想必很多一二线城市的打工人都知道，它就是使用了物联网技术实现了远程定位、锁定、解锁等功能，并且可以收集大量的用户骑行数据。

ofo 利用的物联网技术，包括使用NB-IoT（窄带物联网）作为远程窄带通讯技术，以及NFC（近场通讯）技术。

在最火的物联网爆发元年，我也加入了物联网行业，对物联网抱有很大的想象空间，感觉可以让所有的“物品”都拥有了网络功能，并且都允许人员各种远程收集数据、服务端处理处理数据后存储或者分析、向设备发送新的指令，实现真正的物、网、人一体化。

例如：我们身上的各种“物品”也都能收集并上报我们身体的各个指标数据；可以远程控制家里的门锁、停车场里的汽车、卧室里的空调、手术台上的手术刀等等。

现在2024年回过头来看，大部分都已经实现了，如：智能家居、汽车网络互联、“智能”空调、“智能”门锁…。

其他没实现的，有一些是技术上的可行，但是市场需求并不需要；还有一些是技术水平和人类机器信任度的不足。

当前，我对于物联网的看法是：

因为目前生成式AI的爆发，将推送私有云及边缘等算力基础设施的继续发展，所以AI结合机器人将会迎来很大的一个发展和革新。

而机器人必然离不开物联网技术，利用物联网技术通过传感器收集完各种数据之后，机器人利用这些数据结合AI技术能够分析、规划、反思出符合人类要求的决策和任务，最后通过机器人的外接工具就能够实现更高程度的自动化和协同工作。

这样的技术整合不仅能帮助人类提高工作效率，还可以提高任务完成的准确度。

所以今天跟大家分享的是一些物联网技术知识，供大家参考学习。

物联网是什么？

物联网，英语是Internet of Things，缩写为IoT：指的是我们身边的一切能按约定的协议，将任何物品与互联网连接起来进行信息交换和通信的一种网络形式。

换句话来理解也就是说一切能与网络相连的物品，就都是物联网说的“物”。

物联网就是“物”之间通过连接互联网来共享信息并产生有用的信息，而且无需人为管理就能运行的机制。

物联网的发展

自 20 世纪 90 年代后期起，新型集中式处理围绕着以互联网和 Web 为代表的网络形成了一股发展趋势。这就是 Web 计算的概念。以互联网为媒介，人们可以轻松实现 PC、服务器、移动设备之间的信息交换。

21 世纪初，一个名为“泛在网络”的概念开始受到人们的关注。泛在网络的理念在于使人们能够通过“随时随地”连接互联网等网络来利用多种多样的服务。如：通过智能手机和平板电脑，甚至游戏机、电视机等一些过去无法连接到网络的“物”，就可以随时随地访问互联网。

如今随着宽带的普及，能搭载在机器上的超低功耗传感器投入市场、无线通信技术进步等，都促使除了电脑、服务器和智能手机等传统连接互联网的 IT 相关设备以外，各种各样的“物”也可以连接互联网。

物联网的关键组成部分

物联网的关键组成部分有：感知层（设备层）、网络层、处理层、应用层。

  1. 感知层：这是物联网最底层的部分，主要负责收集和采集信息。常见的感知设备包括传感器、RFID标签、条形码等。
  2. 网络层：这部分负责将感知层收集的数据传输到处理层。网络层包括有线和无线通信网络，如Wi-Fi、蓝牙、Zigbee、LoRaWAN、NB-IoT、2G、LTE 等。
  3. 处理层：处理层对收集到的数据进行处理和分析，包括云计算、边缘计算等技术。这一层还可以包括数据的安全管理和存储。
  4. 应用层：这部分是面向最终用户的，通过各种应用软件和服务来展示处理后的信息，实现特定的功能和服务。

上面的这些设备能连接上网络之后，同时具备两个功能：感测和反馈

设备的作用

感测

设备的感测作用指的是物联网设备通过内置或外接的传感器来收集自身或周围环境或其他物体的信息的过程。

设备自身的信息，包括：钥匙的经纬度信息；机械闹钟的闹钟设定时间…

周围环境或其他物体的信息，包括：手表检测到的人体的心率、血压、血脂；台灯收集到的环境亮度…

而感测有一个重要的电子零件，那就是传感器。传感器能够检测物理世界中的各种信号和变化，并将其转换成数字信号或数据，以便进一步处理和分析。

物联网设备感测的主要类型包括：环境监测、位置与运动监测、声音与图像监测、化学与生物监测、其他特殊用途传感器

不同的感测类型所需的传感器类型也是不一的，比如：

环境监测：温度传感器、湿度传感器、光照传感器、气压传感器。

位置与运动监测：GPS传感器、加速度计、陀螺仪、磁力计。

声音与图像监测：麦克风、摄像头、语音识别传感器。

化学与生物监测：气体传感器、pH传感器、生物传感器。

其他特殊用途传感器：红外传感器、振动传感器、流量传感器。

感测过程的工作原理：

1、信号捕获：传感器检测到物理世界的信号或变化。

2、信号转换：将物理信号转换成电子信号。

3、数据处理：通过微控制器或处理器进行数据的初步处理。

4、数据传输：通过无线或有线网络将数据发送到云端服务器或本地数据处理中心。

5、数据分析与应用：在服务器端或应用层进行数据的深度分析，以提供有价值的信息或触发相应的动作。

反馈

设备的另外一个作用反馈是指：设备根据接收到的信息或指令做出响应的行为。

这种反馈可以是设备自身状态的变化、执行某个动作或是向用户显示信息，所以对应到反馈方式就有：

设备自身状态的变化=>通知

执行某个动作=>控制

向用户显示信息=>可视化
